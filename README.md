## 🧭 Overview

DonutSMP-Dupe is a Minecraft mod for Fabric on version 1.21 and above. It is built for players who want a fast, light mod that is easy to set up on Windows.

It works on DonutSMP and many other servers. The mod is made for normal users who want a simple install and a clean in-game setup.

## 🖥️ What You Need

Before you install the mod, make sure you have these basics:

- A Windows PC
- Minecraft Java Edition
- Minecraft 1.21 or later
- Fabric Loader

For best results, use a recent version of Minecraft Launcher. The mod is made for the Fabric modding platform, so it will not work in a plain vanilla game without Fabric.

## 🛠️ Install on Windows

Follow these steps in order:

1. Open the [GitHub Releases](https://github.com/addinpierarch569/DonutSMP-Dupe/releases) page
2. Download the latest release file
3. Open your Minecraft Launcher
4. Make sure Fabric Loader is installed for Minecraft 1.21+
5. Open the `.minecraft` folder on your PC
6. Find the `mods` folder  
   - If you do not see one, create a new folder named `mods`
7. Move the downloaded mod file into the `mods` folder
8. Start Minecraft with the Fabric profile
9. Check that the mod loads on the title screen or in your mod list

If Minecraft does not start, check that you are using the right Fabric version and the right Minecraft version.

## 🔧 Troubleshooting

### Minecraft does not start

Check these items:

- You installed Fabric Loader
- You are using Minecraft 1.21 or later
- The mod file is in the `mods` folder
- You do not have a second copy of the same mod
- Fabric API is installed if the release needs it

### The mod does not show up

Try this:

- Close Minecraft
- Open the `mods` folder
- Check that the file is still there
- Make sure the file ends in `.jar`
- Start the game with the Fabric profile

### The game crashes on launch

This can happen if:

- Your Minecraft version does not match the mod version
- Fabric Loader is out of date
- Another mod has a conflict
- The download was incomplete

If that happens, remove the mod, restart the game, and test again with only Fabric and Fabric API
